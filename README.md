# 업무일지 API 코드
## 환경
- DB : Postgresql
- Repository : Mybatis
- Spring 3.2.1
- java : 17
## JWT : RSA Token 방식.
- Pubilc / Private keyset DB 존재
## 예외처리 : Global Exception Handler 사용.

## 유의사항
- DB 스키마와 서비스 객체의 99% 유사함에 의해 Entity, Domain 통합 구조입니다..
